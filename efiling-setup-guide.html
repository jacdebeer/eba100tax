eFiling Setup Guide

Complete Guide to SARS eFiling System Setup and Navigation
Updated for 2025 | Version 1.0

üåê Overview

SARS eFiling is the official online platform for submitting tax returns, making payments, and managing your tax affairs. This guide provides step-by-step instructions for setting up and using the eFiling system effectively.

üÜï New in 2025: Express Access

SARS has introduced Express Access for faster login and streamlined filing. This guide covers both traditional and Express Access methods.

Browser: Chrome 90+, Firefox 88+, Safari 14+, or Edge 88+

Mobile Device: For SMS verification and mobile app access

üì± Step 1: Initial Registration

1Access the eFiling Website

  • Navigate to: www.sarsefiling.co.za
  • Click on "Register for eFiling" button
  • Select your entity type: Individual, Business, or Tax Practitioner
[Screenshot: SARS eFiling homepage with Register button highlighted]

2Complete Personal Information

  • ID Number/Passport: Enter your South African ID or passport number
  • Full Names: Must match your ID document exactly
  • Date of Birth: Use DD/MM/YYYY format
  • Contact Details: Active phone number and email address
  • Postal Address: Where SARS correspondence should be sent

3Set Up Security Credentials

  • Username: Choose a memorable username (8-20 characters)
  • Password: Must be 8+ characters with uppercase, lowercase, numbers
  • Security Questions: Choose 3 questions with memorable answers
  • PIN: 5-digit numeric PIN for additional security

‚ö†Ô∏è Password Security Tips

  • Use a unique password not used elsewhere
  • Include special characters (@, #, $, etc.)
  • Avoid personal information (birthdate, phone number)
  • Store securely - don't share with anyone

4Link Tax Numbers

  • Enter your Income Tax reference number
  • Add VAT number if registered for VAT
  • Include PAYE number if you have employees
  • Verify all numbers are correct before proceeding

üîê Step 2: Two-Factor Authentication Setup

1Enable SMS Authentication

  • Verify your mobile number with OTP
  • Choose SMS as your preferred 2FA method
  • Test the SMS functionality
  • Add backup phone number if available

2Set Up Email Verification

  • Check your email for verification link
  • Click the verification link within 24 hours
  • Confirm email address is active and monitored
  • Add SARS email to your safe senders list

üí° Express Access (New 2025 Feature)

Express Access allows faster login using biometric authentication on supported devices:

  • Enable fingerprint or face recognition
  • Requires one-time setup with standard credentials
  • Available on SARS MobiApp and supported browsers
  • Reduces login time from 2-3 minutes to 10-15 seconds

üì± Step 3: SARS MobiApp Installation

1Download the App

  • Android: Download from Google Play Store
  • iOS: Download from Apple App Store
  • Search for "SARS MobiApp" (official app)
  • Verify publisher is "South African Revenue Service"

2Link App to eFiling Account

  • Open the app and select "Link to eFiling"
  • Enter your eFiling username and password
  • Complete SMS verification
  • Set up app-specific PIN or biometric login

üè† Step 4: eFiling Dashboard Navigation

1Dashboard Overview

Your eFiling dashboard provides access to:

  • Returns: Submit and view tax returns

2Key Features Navigation

  • PAYE Returns (EMP201, EMP501)
  • Import/Export declarations

Work Online Features

  • Request for tax clearance certificates
  • Update contact and banking details
  • Register for additional tax types
  • Submit supporting documents

üîß Troubleshooting Common Issues

‚ö†Ô∏è Common Problems & Solutions

Login Issues

  • Forgotten password: Use "Forgot Password" function
  • Account locked: Contact SARS support after 3 failed attempts
  • SMS not received: Check network coverage, try alternative number

Technical Problems

  • Browser compatibility: Clear cache, try different browser
  • Session timeout: Save work frequently, extend session when prompted
  • File upload errors: Check file size (max 5MB) and format

Account Issues

  • Missing tax numbers: Contact SARS to link additional numbers
  • Incorrect information: Use "Update Details" function
  • Representative access: Complete appointment forms
